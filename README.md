# TalentScout AI Hiring Assistant

TalentScout is a Streamlit-based AI chatbot that performs initial candidate screening using OpenAI.

## Features
- Conversational hiring assistant
- Collects candidate information
- Generates technical questions based on tech stack
- Context-aware AI responses
- GDPR-compliant (session-only)

## Tech Stack
- Python
- Streamlit
- OpenAI GPT

## Deployment
Deployed on Streamlit Community Cloud using secure API secrets.
